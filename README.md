# Сборщик изображений для датасета классификации медведей

Автоматизированный инструмент веб-скрейпинга, написанный на Python, который собирает изображения полярных и бурых медведей с Яндекс.Картинок для целей машинного обучения и классификации.

- [Предварительные требования](#предварительные-требования)
- [Установка](#установка)

## Возможности

- **Сбор изображений нескольких классов**: Автоматически собирает изображения для нескольких категорий (полярный медведь, бурый медведь)
- **Надежная обработка ошибок**: Обрабатывает сетевые ошибки, поврежденные изображения и отсутствующие файлы
- **Отслеживание времени**: Измеряет и отображает время выполнения для каждой фазы
- **Фильтрация качества**: Отсеивает маленькие изображения и не-изображения
- **Мониторинг прогресса**: Обновления в реальном времени во время сбора и загрузки
- **Предотвращение дубликатов**: Обеспечивает уникальность собранных URL
- **Поддержка форматов**: Поддерживает несколько форматов изображений (JPG, PNG, GIF, BMP, WEBP)
- **Защита от обнаружения**: Обходит распространенные механизмы обнаружения ботов
- **Отчеты статистики**: Генерирует подробную статистику и сохраняет в файл

## Предварительные требования

- **Python 3.7+**
- **Браузер Chrome** на вашей системе
- **ChromeDriver**, соответствующий вашей версии Chrome
- **Необходимые Python-пакеты**
